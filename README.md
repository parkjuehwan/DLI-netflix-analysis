# 넷플릭스 국가별 컨텐츠 제작 순위 분석

`netflix_titles.csv`를 기반으로 국가별 넷플릭스 컨텐츠 제작 순위를 집계하고, 결과를 인터랙티브 HTML로 시각화하는 프로젝트입니다.

## 파일 구성

| 파일 | 설명 |
|---|---|
| `netflix_titles.csv` | 원본 데이터셋 |
| `netflix_analysis.ipynb` | 데이터 전처리 및 순위 집계 노트북 |
| `ranking_data.json` | 노트북 집계 결과를 HTML에서 쓰기 위해 내보낸 JSON |
| `netflix_country_ranking.html` | 검색 + 도넛 차트 3종으로 순위를 보여주는 시각화 페이지 |
| `prompt.txt` | 분석/시각화 요구사항 명세 |

## 데이터 전처리

1. `title`, `country` 컬럼만 남기고 나머지는 삭제, `country`가 결측인 행 삭제
2. `country`를 쉼표 기준으로 분리해 참여 국가 리스트로 변환 (끝에 쉼표가 붙어 생기는 빈 문자열은 제거)
3. 참여 국가 수를 `co_production` 컬럼으로 저장
4. 공동제작 컨텐츠는 참여 국가 수만큼 행을 분리(explode)
5. 공동제작 행은 국가명 뒤에 `_co_production` 접미사를 추가 (예: `Korea_co_production`)

## 순위 집계 3가지 방식

| 방식 | 설명 |
|---|---|
| ① 단독제작만 인정 | 공동제작 컨텐츠는 집계에서 제외하고 단독제작 편수만 카운트 |
| ② 공동제작 전체 인정 | 공동제작도 참여 국가마다 1편으로 온전히 인정 |
| ③ 공동제작 분할 인정 | 공동제작은 참여 국가 수(`co_production`)로 나눠 `1/n`편씩 인정 |

## 시각화 (`netflix_country_ranking.html`)

- 브라우저에서 파일을 직접 열면 됩니다 (별도 서버 불필요).
- 상단 검색창에 국가명을 영어로 입력하면 3가지 방식 각각의 순위를 확인할 수 있습니다.
- 3개의 도넛 차트가 각 방식별 1~10위를 보여주고, 11번째 조각(회색 "기타")은 나머지 모든 국가의 합계입니다.
- 각 차트 아래 "표로 보기"를 누르면 전체 국가 순위를 표로 볼 수 있습니다.

## 알려진 데이터 이슈

원본 CSV에 `"United States,"`처럼 끝에 쉼표가 붙은 행이 일부 있어, 분리 시 빈 문자열이 국가로 잘못 집계될 수 있었습니다. 전처리 단계에서 빈 문자열을 제거해 `co_production` 값이 부풀려지지 않도록 처리했습니다.
